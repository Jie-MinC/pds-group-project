### Folder Structure

| Folder        | Description                       |
| ------------- | --------------------------------- |
| data          | Datasets for analysis.            |
| model         | Reusable machine learning models. |
| notebook      | R Markdown file for analysis.     |
| presentation  | R Presentations files.            |
| visualization | Visualization files.              |
| web           | Shiny app code.                   |
